# Wireshark Network Traffic Analysis

## Objective
The objective of this project was to analyze different types of network traffic using Wireshark.

---

## Tools Used
- Wireshark
- Windows OS

---

## Traffic Captured

### 1. DNS Traffic
File:
google_youtube_dns_capture.pcapng

Observations:
- DNS queries were generated while visiting Google and YouTube.
- DNS translated domain names into IP addresses.
- Observed query and response packets.

---

### 2. HTTP Traffic
File:
neverssl_http_traffic_analysis.pcapng

Observations:
- HTTP traffic was readable in plain text.
- GET requests and host information were visible.
- User-Agent and website details could be inspected.

---

### 3. HTTPS/TLS Traffic
File:
google_https_tls_traffic_analysis.pcapng

Observations:
- TLS traffic was encrypted.
- Application data could not be read directly.
- HTTPS provided secure communication.

---

## Key Learning Outcomes
- Learned packet capture using Wireshark
- Understood DNS request/response behavior
- Compared HTTP and HTTPS traffic
- Identified encrypted TLS communication
- Practiced packet filtering using Wireshark filters

---

## Conclusion
This project helped develop foundational network traffic analysis and cybersecurity investigation skills using Wireshark.
